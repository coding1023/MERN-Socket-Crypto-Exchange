# Crypto Exchange Widget

This widget can be used to display
several currency exchanges rates and it will allow the user to exchange USD  for
Crypto  The application consists of a React front-end client and a Node back-end
service (Express.js).

## Installation


Some basic Git commands are:

$ git clone https://github.com/redacreltd/Claudio-Sapia.git
$ cd project
$ npm install
